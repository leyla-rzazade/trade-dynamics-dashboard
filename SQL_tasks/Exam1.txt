Task2
Şirkətdə maaş artım edilməsi ilə bağlı qərar imzalanıb.Belə ki, maaşı 6000-dən
kiçik bərabər 
və ya  IT_PROG vəzifələrində işləyən əməkdaşlara 13 faiz artım, 
maaşı 6000 dən böyük əməkdaşlara isə
10 faiz artım olunmalıdır. Buna uyğun select yazın.
Nəticə yekun artim məbləği əsasında çoxdan aza düzülsün.
Sətir sayı:107

select 
LAST_NAME,
FIRST_NAME,
salary,
JOB_ID,
case 
    when salary<=6000 or JOB_ID='IT_PROG' then (salary*1.13)
    when salary>6000 then (salary*1.1)
    end as artim
 from hr.EMPLOYEES;

Task 3
Locations cədvəlində ya postal_code,ya city, ya da state_province sutunlarindan 
biri boş olan sətirləri gətirin.
Ekrana location_id,street_address ve nn_address sutunlarını çıxarın
nn_address sütunu belə formalaşır: postal_code,state_province,city sütunlarından qeyd olunan ardıcıllıqda 
ilk hansında məlumat varsa o gəlsin ,əgər hər 3 - ü də boşdursa 'məlumat tapılmadı' 
qeyd olunsun.
Sətir sayı:6
select 
LOCATION_ID,
STREET_ADDRESS,
case 
    when POSTAL_CODE is not null then POSTAL_CODE
    when STATE_PROVINCE is not null then STATE_PROVINCE
    when CITY is not null then CITY
    else 'melumat tapilmadi'
    end  as nn_address
 from hr.LOCATIONS
where POSTAL_CODE is null or STATE_PROVINCE is null or CITY is null

Task 4
Soyadında rast gəlinən ilk b hərfi 3-cü hərfdən sonra  olan sətirlər üçün adın 
uzunlugu,ad və soyad məlumatlarını çıxaran SQL yazın
Sətir sayı:4
select 
FIRST_NAME,
LAST_NAME,
length(FIRST_NAME) as uzunluq,
instr(lower(LAST_NAME),'b') as find
 from hr.EMPLOYEES
 where instr(lower(LAST_NAME),'b')>3

Task 6
50-ci department-də işləyən hər bir işçinin 
adını, soyadını, maaşını,ilk maaş artımı olacağı tarixi tapın.
Maaş artımı tarixi belə hesablanır:
İşçinin işə girdiyi tarixdən 6 ay sonrakı tarixin düşdüyü ayın son günü.
Sətir sayı:45
select 
first_name,
last_name,
salary,
HIRE_DATE,
DEPARTMENT_ID,
last_day(add_months(HIRE_DATE,6)) as "maas artim tarixi"
from hr.EMPLOYEES
where DEPARTMENT_ID=50
